## Getting Started

### Clone the repository

```bash
git clone https://github.com/Soleman-Kamal/Balancely-ERP.git
```

### Navigate to the project

```bash
cd Balancely-ERP
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

---

## Build for Production

```bash
npm run build
```
